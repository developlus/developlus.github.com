============================
 Tinkerer(Sphinx) on GitHub
============================

Web UIで執筆するCMSやブログシステムに耐えられなくて，さまざまなドキュ
メント管理ツールを試した結果，ReST（reStructured Text）にたどり着く．

ReSTを整形するツールである「Sphinx」，さらにSphinxを利用するブログ構築
を支援する「Tinkerer」を使って，今後のドキュメントを管理していく．

Tinkerer
========

http://tinkerer.me/

コマンド一つでブログのテンプレートの作成，ドキュメントのビルドを実行す
ることが可能です．生成されるドキュメントは静的なものなので，どこにでも
引っ越し可能です．

GitHub
======

https://github.com/

言わずとしれた分散リポジトリ管理システムです．
無料で個人のリポジトリを作成できることから巷でも人気のようですが，自分
で書いたソースコードをインターネットな世界に晒す勇気がなくて躊躇してい
ました．お仕事のソース管理はSubversionで十分でしたし．

しかし，Tinkerer関連の情報を集めていくうちに，GitHubも使いようによって
はGitHub上でブログを書いて置けるとか・・・しかも，リポジトリなのでどこ
からでも編集可能ということで採用してみることにしました．

GitHub上をマスターとして，個人のサーバ上にリポジトリのコピーをおいてお
けば，編集は一箇所で済むし，バックアップも不要．一石二鳥です．
